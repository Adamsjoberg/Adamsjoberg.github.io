Android application UI design

This is an ongoing project, far from finished, which me and a group of software developers started to create a few months back. The concept of this application is to make spontaneous events easier to execute. Way too often do we encounter situations where we text, call and message in between friends in order to organize a simple event like playing soccer on a sunny day or just meet for a coffee. With ReadyApp, there is no need for that amount of communication and hesitation, you can simply create an event, invite your friends and send a ready-check. All the participants are able to see who is invited, who is "ready" and who is not.

Anyhow, these are a few of the screens created by me and I as you can see a lot of the architecture is not finished yet. I've been doing all the UI/UX by myself and worked closely with the people who produce the code by explaining the UX of the application
